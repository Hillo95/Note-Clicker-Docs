NoteClicker Version 0.1.0.0

NoteClicker is a work in progress clicker game based off of the popular web game Cookie Clicker.

Play the most recent version of NoteClicker at noteclicker.netlify.app.

The following is a list of all updates up to this point, released and unreleased.

The first number is the stage of the game, four stages are planned, the second number is the primary update, the third number is any change to the update, the fourth number is for a bugfix.

Release 0.0.0.0

Added the clickable note in it's simpliest form as well as five shop items, clicking effects, and information about shop items.

Update 0.1.0.0

Added a simple saving system using local storage on the browser allowing you to save, load, and reset your data on your device, this update also added autosaving every ten seconds and autoloading on page load.

Patch 0.1.0.1

Fixed a client issue involving an error in the data loading system.

Update 0.2.0.0

Added five new shop items and introduced the shop level.

Patch 0.1.0.2

Fixed a code error affecting the saving system.
